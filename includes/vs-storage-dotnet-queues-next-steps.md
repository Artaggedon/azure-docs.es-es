Ahora que está familiarizado con los aspectos básicos del almacenamiento de colas de Azure, use estos vínculos para obtener más información acerca de tareas de almacenamiento más complejas.

* Consulte la documentación de referencia del servicio Cola en la referencia [Biblioteca de cliente de Azure Storage para .NET](/dotnet/api/overview/azure/storage) para obtener información detallada acerca de las API disponibles.
* Obtenga más información sobre el uso del almacenamiento en cola en [Introducción al Almacenamiento en cola de Azure mediante .NET](../articles/storage/queues/storage-dotnet-how-to-use-queues.md)
* Para aprender a simplificar el código que escriba para trabajar con Almacenamiento de Azure, consulte [Qué es el SDK de Azure WebJobs](https://github.com/Azure/azure-webjobs-sdk/wiki)
* Consulte más guías de características para obtener información acerca de otras opciones del almacenamiento de datos en Azure.
  * Para trabajar con las tablas de Azure Storage, consulte [Introducción a Azure Table Storage mediante .NET](../articles/cosmos-db/tutorial-develop-table-dotnet.md).
  * Para trabajar con los blobs de Azure Storage, consulte [Introducción Azure Blob Storage mediante .NET](../articles/storage/blobs/storage-quickstart-blobs-dotnet.md).
  * Para almacenar datos relacionales, consulte [Conexión a SQL Database mediante .NET (C#)](../articles/azure-sql/database/connect-query-dotnet-visual-studio.md).
