El SDK de Azure Spatial Anchors se debe configurar en <a href="https://unity.com/releases/2019-lts" target="_blank">Unity 2019.4 (LTS)</a> mediante los siguientes paquetes:
- AR Foundation: <a href="https://docs.unity3d.com/Packages/com.unity.xr.arfoundation@3.1/manual/index.html" target="_blank">3.1.3</a>
- Windows Mixed Reality: <a href="https://docs.unity3d.com/Packages/com.unity.xr.windowsmr.metro@4.2/manual/index.html" target="_blank">4.2.1</a>
- Complemento ARCore XR (para la compatibilidad con dispositivos Android): <a href="https://docs.unity3d.com/Packages/com.unity.xr.arcore@3.1/manual/index.html" target="_blank">3.1.3</a>
- Complemento ARKit XR (para la compatibilidad con dispositivos iOS): <a href="https://docs.unity3d.com/Packages/com.unity.xr.arkit@3.1/manual/index.html" target="_blank">3.1.3</a>

> [!NOTE]
> El SDK de ASA no admite actualmente la <a href="https://docs.unity3d.com/Manual/XRPluginArchitecture.html" target="_blank">plataforma de complementos Unity XR</a>
