> [!IMPORTANT]
> Si ve la siguiente advertencia, debe actualizar el software de los dispositivos antes de continuar:<br>
> 
> *Uno o varios dispositivos StorSimple utilizan una versión antigua del software. La actualización más reciente disponible para TLS 1.2 es una actualización obligatoria y debe instalarse inmediatamente en estos dispositivos. TLS 1.2 se usa para toda la comunicación de Azure Portal y sin esta actualización, el dispositivo no podrá comunicarse con el servicio StorSimple.*
>
