SMB multicanal para los recursos compartido de archivos de Azure solo está disponible en las siguientes regiones:

| Región geográfica | Región de Azure |
|-------------|-------------------|
| Australia | Este de Australia |
| Brasil | Sur de Brasil |
| Canadá | Centro de Canadá y Este de Canadá |
| Francia | Centro de Francia |
| Alemania | Centro-oeste de Alemania |
| India | Centro de la India e India occidental |
| Japón | Japón Oriental, Japón Occidental |
| Corea | Centro de Corea del Sur y Oeste de Corea |
| Emiratos Árabes Unidos | Norte de Emiratos Árabes Unidos |
| Reino Unido | Sur de Reino Unido y Oeste de Reino Unido |
| Estados Unidos | Centro-norte de EE. UU., Centro-sur de EE. UU. y Centro-oeste de EE. UU. |
