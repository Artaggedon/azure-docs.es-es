## <a name="go-to-your-api-management-instance"></a>Vaya a la instancia de API Management.

1. En Azure Portal, busque y seleccione **Servicios de API Management**.

    :::image type="content" source="media/api-management-navigate-to-instance/view-apim-1.png" alt-text="Seleccionar servicios de API Management":::

1. En la página de **servicios API Management**, seleccione la instancia de API Management.

    :::image type="content" source="media/api-management-navigate-to-instance/view-apim-2.png" alt-text="Seleccionar su instancia de API Management":::
