# <a name="azure-sql-edge-security-partners"></a>Asociados de seguridad de Azure SQL Edge

En este artículo se resaltan las empresas asociadas de Microsoft con soluciones de seguridad para proporcionar soluciones de seguridad de un extremo a otro para las implementaciones de IoT Edge con Azure SQL Edge.

## <a name="security-partners"></a>Asociados de seguridad
 
| Asociado| Descripción | Vínculos |
|-----|-----|-----|
|![DH2i](media/resources/dh2i-logo.png)|DH2i toma un nuevo enfoque innovador para la conectividad de redes, ya que proporciona a las organizaciones su infraestructura de TI Always On y siempre segura de perímetro definido de software (SDP). DxOdyssey para IoT la extiende a los dispositivos perimetrales, lo que permite un acceso sin problemas desde los dispositivos perimetrales hasta el centro de datos y la nube. Este módulo de SDP se ejecuta en cualquier dispositivo IoT de un contenedor en la arquitectura de x64 y arm64. Una vez habilitadas, las organizaciones pueden crear túneles de nivel de aplicación privados y seguros entre dispositivos y centros de conectividad sin el requisito de una VPN o sin exponer puertos abiertos y públicos. Este módulo de SDP está diseñado específicamente para casos de uso de IoT en los que los dispositivos perimetrales deben comunicarse con otros dispositivos, recursos, aplicaciones o nubes. Requisitos mínimos de hardware: Sistema operativo Linux x64 y arm64, 1 GB de RAM y 100 MB de almacenamiento.| [Sitio web](https://dh2i.com/) [Marketplace](https://ms.portal.azure.com/#blade/Microsoft_Azure_Marketplace/MarketplaceOffersBlade/selectedMenuItemId/home) [Documentación](https://dh2i.com/dxodyssey-for-iot/) [Soporte](https://dh2i.com/support/)
