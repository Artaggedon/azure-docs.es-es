# <a name="advanced-development-with-azure-percept"></a>Desarrollo avanzado con Azure Percept
Con Azure Percept, los desarrolladores de software y los científicos de datos pueden usar flujos de trabajo de código avanzados para la administración del ciclo de vida de inteligencia artificial. A través de una biblioteca de código abierto cada vez mayor, pueden usar ejemplos para empezar a trabajar con su recorrido de desarrollo de inteligencia artificial y crear soluciones listas para producción.

## <a name="get-started-with-the-advanced-development-tutorials"></a>Primeros pasos con los tutoriales de desarrollo avanzado

Este [cuaderno de Jupyter Notebook](https://github.com/microsoft/Project-Santa-Cruz-Private-Preview/blob/main/Sample-Scripts-and-Notebooks/Official/Machine%20Learning%20Notebooks/Transferlearningusing_SSDLiteV2%20Model.ipynb) realiza el aprendizaje por transferencia mediante un modelo de TensorFlow preentrenado en AzureML (MobileNetSSDV2Lite) en Python con un conjunto de datos personalizado para detectar cuencos. Hay dos opciones de tutorial que le guiarán en el trabajo con el cuaderno para entrenar e implementar el detector de cuencos en el proyecto Santa Cruz DevKit:

- [Desarrollo en la nube](./advanced-development-cloud.md): en este tutorial, ejecutará el cuaderno en el [portal de Azure Machine Learning](https://ml.azure.com) con una instancia de proceso remota.

- [Desarrollo local](./advanced-development-local.md): en este tutorial, ejecutará el cuaderno localmente en VS Code con una instancia de proceso remota.
