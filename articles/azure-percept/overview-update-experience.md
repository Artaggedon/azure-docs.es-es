# <a name="the-azure-percept-dk-update-experience"></a>Experiencia de actualización de Azure Percept DK

Con Azure Percept DK, tiene dos opciones para actualizar el sistema operativo y el firmware del kit de desarrollo: transmisión por vía inalámbrica (OTA) o a través de USB. La actualización por vía inalámbrica es la mejor manera de mantener los dispositivos actualizados, mientras que la actualización mediante USB es una buena opción para cuando no es posible la vía inalámbrica o cuando se desea restablecer el dispositivo con los valores de fábrica. Para asegurarse de que aprovecha el método de actualización que mejor se adapta a su caso, hemos elaborado una colección de guías de procedimientos que le ayudarán. 

- [Configuración de Azure IoT Hub para la implementación de actualizaciones de forma inalámbrica en Azure Percept DK](./how-to-set-up-over-the-air-updates.md)
- [Actualización de Azure Percept DK de forma inalámbrica](./how-to-update-over-the-air.md)
- [Actualización de Azure Percept DK mediante USB](./how-to-update-via-usb.md)
