# <a name="assign-user-roles-and-permissions"></a>Asignación de roles de usuario y permisos

**Roles adecuados**

- Propietario
- Manager

Para acceder a funcionalidades relacionadas con el marketplace o su cuenta de desarrollador, debe tener asignado uno o varios de los siguientes roles. Cada rol tiene un conjunto específico de permisos para realizar determinadas funciones dentro de la cuenta.

| Role | Descripción |
| ------------ | ------------- |
| Propietario | Cuando se crea una cuenta de publicador, se asigna un rol de propietario al partner.<br><br>Detalles del permiso:<br><br>&#10004;&#160;&#160;Propiedad total de la cuenta.<br>&#10004;&#160;&#160;Agregar cualquier tipo de usuarios.<br>&#10004;&#160;&#160;Poder completar información fiscal y de pago.<br>&#10004;&#160;&#160;Ver resúmenes e informes de comentarios.<br>&#10004;&#160;&#160;Inscribir la cuenta de publicador en cualquier otro programa para desarrolladores. |
| Manager | &#10004;&#160;&#160;Administrar usuarios, roles y permisos de la cuenta de publicador.<br>&#10004;&#160;&#160;Administrar inquilinos.<br>&#x2718;&#160;&#160;Acceder a la configuración fiscal y de pago. |
| Desarrollador | &#10004;&#160;&#160;Cargar paquetes.<br>&#10004;&#160;&#160;Enviar aplicaciones y componentes.<br>&#10004;&#160;&#160;Ver informes de uso para la información de telemetría.<br>&#x2718;&#160;&#160;Acceder a información financiera o a la configuración de la cuenta. |
| Colaborador empresarial | &#10004;&#160;&#160;Acceder a información financiera.<br>&#10004;&#160;&#160;Ver detalles de precios.<br>&#x2718;&#160;&#160;Crear o enviar nuevas aplicaciones y complementos. |
| Colaborador financiero | &#10004;&#160;&#160;Ver informes de pago.<br>&#10004;&#160;&#160;Administrar perfiles fiscales y de pago.<br>&#x2718;&#160;&#160;Realizar cambios en las aplicaciones o la configuración. |
| Vendedor | &#10004;&#160;&#160;Responder a las reseñas de clientes.<br>&#10004;&#160;&#160;Ver informes no financieros.<br>&#x2718;&#160;&#160;Realizar cambios en las aplicaciones o la configuración. |
|||

> [!NOTE]
> En el caso del programa Marketplace comercial, no se utilizan los roles Administrador global, Colaborador empresarial, Colaborador financiero y Vendedor. Asignar estos roles a los usuarios no tiene ningún efecto. Solo los roles Administrador y Desarrollador conceden permisos a los usuarios.

Para más información sobre la administración de roles y permisos en otras áreas del Centro de partners, como Azure Active Directory (AD), Cloud Solution Provider (CSP), Control Panel Vendor (CPV), usuarios invitados o Microsoft Partner Network (MPN), consulte [Asignación de roles y permisos de usuario en el Centro de partners](/partner-center/permissions-overview).

> [!NOTE]
> Cualquier actividad de asignación de roles o administración de usuarios realizada en estas líneas estará en el contexto de la cuenta en la que se encuentra. Consulte la sección acerca de cómo cambiar entre vendedores si necesita administrar una cuenta diferente.
