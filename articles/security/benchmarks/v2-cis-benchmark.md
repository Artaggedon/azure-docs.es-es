# <a name="mapping-of-azure-security-benchmark-v2-and-cis-microsoft-azure-foundations-benchmark"></a>Asignación de Azure Security Benchmark v2 y CIS Microsoft Azure Foundations Benchmark

CIS Microsoft Azure Foundations Benchmark es la guía de seguridad que proporciona Center for Internet Security para establecer una configuración de línea de base segura para Azure. El ámbito del punto de referencia es establecer el nivel base de seguridad mientras se adopta la nube de Azure. Este punto de referencia está en consonancia con [Azure Security Benchmark v2](overview.md).

Se proporciona una asignación entre la [versión más reciente (1.3.0) de CIS Microsoft Azure Foundations Benchmark](https://www.cisecurity.org/benchmark/azure/) y [Azure Security Benchmark v2](overview.md). Esto le va a ayudar a comprender cómo se alinean las recomendaciones de ambos puntos de referencia. Puede descargar la asignación completa [aquí](https://github.com/MicrosoftDocs/SecurityBenchmarks/blob/master/Azure%20Security%20Benchmark/2.0/asb_v2_to_cis_microsoft_azure_foundations_benchmark_v1.3.0.xlsx).
