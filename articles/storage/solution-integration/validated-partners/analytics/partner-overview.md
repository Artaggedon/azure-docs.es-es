# <a name="azure-storage-analytics-and-big-data-partners"></a>Asociados de macrodatos y análisis con Azure Storage

En este artículo se destacan las empresas asociadas de Microsoft que se integran con Azure Data Lake Storage. Estas soluciones de asociados abarcan cargas de trabajo modernas de almacenamiento de datos, análisis avanzado y análisis en tiempo real, entre otras. Estos asociados aprovechan el [espacio de nombres jerárquico](../../../blobs/data-lake-storage-namespace.md) de Azure Storage para optimizar su solución y ejecutarla de forma eficaz en Azure.

| ------- | ----------- | -------------------- |
