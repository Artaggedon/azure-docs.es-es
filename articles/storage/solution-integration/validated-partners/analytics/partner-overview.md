# <a name="azure-storage-analytics-and-big-data-partners"></a>Asociados de Azure Storage Analytics y Big Data

En este artículo se destacan las empresas asociadas de Microsoft que se integran con Azure Data Lake Storage. Se ocupan de cargas de trabajo que incluyen almacenamiento de datos moderno, análisis avanzado, análisis en tiempo real, etc. Estos asociados aprovechan las ventajas del espacio de nombres jerárquico de Azure Storage para optimizar su solución y ejecutarla de forma eficaz en Azure.
