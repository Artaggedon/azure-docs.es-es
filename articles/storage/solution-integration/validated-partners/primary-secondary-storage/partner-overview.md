# <a name="azure-storage-partners-for-primary-and-secondary-storage"></a>Asociados de Azure Storage para el almacenamiento principal y secundario

En este artículo se destacan las empresas asociadas de Microsoft que ofrecen una solución local, NAS o SAN, en Azure o en una solución híbrida que usa Azure Storage como una opción de nivel rentable. Estas soluciones pueden permitir a los clientes usar la misma solución en cualquiera de sus entornos.
