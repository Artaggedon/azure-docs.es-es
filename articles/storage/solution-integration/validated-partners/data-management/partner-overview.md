# <a name="azure-storage-data-governance-management-and-migration-partners"></a>Asociados de gobernanza, administración y migración de datos de Azure Storage

En este artículo se destacan las empresas asociadas de Microsoft integradas con Azure Storage que mejoran las funcionalidades generales de administración de datos del cliente. Estas soluciones de asociados pueden admitir la evaluación y los informes de almacenamiento, la migración independiente de la plataforma, la replicación, la organización de la nube por niveles o la gobernanza de los datos.
