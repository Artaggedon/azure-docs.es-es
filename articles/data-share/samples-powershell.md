# <a name="azure-powershell-samples-for-azure-data-share"></a>Ejemplos de Azure PowerShell para Azure Data Share

En la tabla siguiente se incluyen vínculos a scripts de ejemplo de Azure PowerShell para Azure Data Share.

|Ejemplos de Powershell|Descripción|
|---|---|
|[Creación de una cuenta de recurso compartido de datos](scripts/powershell/create-new-share-account-powershell.md)| Este script de PowerShell crea una cuenta de recurso compartido de datos. |
|[Creación de un recurso compartido de datos](scripts/powershell/create-new-share-powershell.md)| Este script de PowerShell crea un recurso compartido de datos. |
|[Adición de conjuntos de datos a un recurso compartido](scripts/powershell/add-datasets-powershell.md)| Este script de PowerShell agrega un conjunto de datos de blob a una cuenta de recurso compartido de datos existente. |

|[Creación de una invitación de recurso compartido de datos](scripts/powershell/create-share-invitation-powershell.md)| Este script de PowerShell de ejemplo crea una invitación para proporcionar acceso a un recurso compartido de datos existente. |
|[Visualización de las invitaciones de recurso compartido enviadas](scripts/powershell/view-sent-invitations-powershell.md)| Este script de PowerShell de ejemplo recupera una lista de las invitaciones enviadas desde un recurso compartido de datos existente. |

|[Creación y visualización de desencadenadores de instantáneas](scripts/powershell/create-view-trigger-powershell.md)| Este script de PowerShell de ejemplo crea los desencadenadores de instantáneas de un recurso compartido.
